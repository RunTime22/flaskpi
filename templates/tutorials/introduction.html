Introduction

The most of the software are developes as web services or SaaS product. So web development is very popular now. I didn't find any tutorial wich describes web development from the first line of code to deployment to the server. So I decided to create one.

I'm learning Flask now and want to share my knowledge with everybody. I hope, this project will be a good tutorial and web-site development process example.

Raspberry Pi is very compact and pretty cheap computer. Of course, you can use some virtual machine to learn but VMs isn't so portable as Raspbery Pi. You can you your Raspberry Pi as a simple home server. It will be quiet (no coolers or mechanic parts), energy saving server. And you don't need to get powered on a big PC.

This version of the raspberypi.com is only prototype of the real website (look to the 'prototype' branch on GitHub). Step-by-step I'll create new features and share sources and tutorial with you. You are welcome to contribute code, tutorials or fixes to me. Just fork this project on GitHub and create Pull Request.
